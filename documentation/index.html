Import Blog



A Jam stack Blog via angular and contentful, where i use contentful as a CMS for angular frontend. This blog uses contentful's content delivery API to fetch blogs for the frontend.

Installation

(Back to top)

  1. Clone the repo
     git clone https://github.com/obeymyshinnyrod/import-blog/
  2. Install the NPM packages
        npm install
  3. You need to get your space set up on contentful signup here
  4. Set up your content model at Content Model tab
  5. Add your blog posts on Content tab
  6. Now set up the enviroment.ts file
  7. Visit the documentation for more details
  8. To run the app
        ng serve
  9. The project is live on "http://localhost:4200/"

Development

(Back to top)

After the Installation phase you can modify and run the code on your local host for that you need to have basic knowledge of Angular.

Ideally you can pick up issues from the issues tab in the repo and work on them or if you wanna add an new feature you can push the code on a new branch with describing the feature and making sure there are now merge errors.
